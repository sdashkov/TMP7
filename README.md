# Разработка программных средств поиска и анализа киберугроз  
## 7 семестр

В данном репозитории собраны все практические работы, выполненные в рамках дисциплины **«Разработка программных средств поиска и анализа киберугроз»** в 7 семестре.

Репозиторий создан с целью структурирования учебных материалов и демонстрации практических навыков работы с различными моделями машинного обучения.

---

## Содержание

В репозитории представлено **8 Jupyter Notebook файлов (`.ipynb`)**, каждый из которых соответствует отдельной теме:

1. Кластеризация  
2. Метод *k-ближайших соседей (KNN)* для классификации  
3. Метод *k-ближайших соседей (KNN)* для регрессии  
4. Деревья решений  
5. Регуляризация деревьев решений  
6. Случайный лес для задач регрессии  
7. Градиентный бустинг для задач регрессии  
8. Градиентный бустинг для задач классификации  

---

## Используемые подходы и навыки
